## What I learned :  interview-flavoured

Built two production-grade API clients using Python's `requests` library against public REST APIs (SMARD Bundesnetzagentur data and Open-Meteo). Learned the difference between ETL (notebook 01) and EDA (notebook 02). Implemented a physics-based solar irradiance model using pvlib's Ineichen-Perez clear-sky algorithm, establishing a principled baseline before any machine learning. The residual decomposition, *actual = physics + residual*, is the architectural spine of the whole project.
